# nucleo-f446re-hc-sr04-servo-motor
Control servo motor and hc-sr04 using STM32F446RE

Connect:
- PA0 (A0) --> Orange cable Servo Motor
- PA1 (A1) --> Trigger HC-SR04
- PA4 (A2) --> Echo HC-SR04
- PB7 --> SDA LCD
- PB6 --> SCL LCD
